YouTube Transcript Summarizer
YouTube Transcript Summarizer is a simple yet powerful tool that automatically fetches transcripts from YouTube videos and generates concise summaries. Whether youâ€™re researching, studying, or just short on time, this tool helps you quickly grasp the key points of any video without watching the whole thing.

Features
Fetches transcripts directly from YouTube videos (where available).

Generates clear, concise summaries.

Supports multiple languages (if transcripts are available).

Easy to run locally with minimal setup.

How It Works
You provide the YouTube video URL.

The tool fetches the transcript.

It processes and summarizes the content for you.

Requirements
Python 3.8+

Required dependencies listed in requirements.txt
